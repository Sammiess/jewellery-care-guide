# WooCommerce Jewellery Care Link

A lightweight WordPress snippet that adds a jewellery care link below WooCommerce product pages.

## What it does
This simple function automatically displays a small “Care tips for your jewellery” link under every product. It helps customers understand how to maintain their pieces and keeps them engaged on your site.

## Features
- No settings required  
- Lightweight and fast  
- Works on all WooCommerce product pages  
- Uses a standard WooCommerce hook  

## Installation
1. Copy the PHP file into your theme’s `functions.php` or a custom plugin  
2. Adjust the URL to your own jewellery care guide if needed  

## Example link
https://yazzys.com/jewellery-care-guide/

## Use case
Ideal for jewellery stores that want to improve customer trust and provide added value directly on product pages.
